### Cara Menggunakan

1. Clone repository ini:
   ```bash
   git clone https://gitlab.com/storyku/front-end-storyku
   ```
2. Masuk ke folder proyek:
   ```bash
   cd front-end-storyku
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Jalankan aplikasi:
   ```bash
   npm start
   ```
